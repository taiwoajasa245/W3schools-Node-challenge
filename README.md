# Simple Node.js Web Server

This is a simple Node.js project that hosts a static HTML file using an node server. The server is set up to serve the `index.html` file .

## Prerequisites

Before running the server, make sure you have Node.js installed on your machine. You can download it from [Node.js website](https://nodejs.org/).

## Setup

1. **Clone this repository:**

   ```bash
   git clone https://github.com/taiwoajasa245/W3schools-Node-challenge.git

2. **Navigate to the project directory:**

   ```bash
   cd W3schools-Node-challenge

3. **Start up your server**

   ```bash
   node server
